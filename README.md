# Yun Tao's Portfolio

My personal portfolio website showcasing my work experience, projects, and technical skills.

## Live Site

Visit: [https://y83tao.github.io/portfolio/](https://y83tao.github.io/portfolio/)

## About

A minimalistic blue-themed portfolio website built with React and Framer Motion, featuring:
- Animated space background with stars and planets
- Smooth page transitions
- Fully responsive design
- Work experience timeline
- Project showcases with live production indicators
- Technical skills with icons

## Tech Stack

- React 18
- Vite
- Framer Motion
- React Router
- React Icons
- CSS3

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm

### Installation

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Deploy to GitHub Pages
npm run deploy
```
